	Bootstrap Spinners
------------------------	
1) Bootstrap 4 introduces the new spinner component that you can use to show the loading state in your projects. Spinners are typically loading icons and they're built only with HIM and CSS. You can easily customize the appearance, alignment, and sizing of the spinners with the pre-defined utility classes. 

2) To create a spinner or loading indicator, you can use the .spinner-border class.

3) You can use the text color utility classes to customize the color of spinners.

4) You can also create growing spinners that repeatedly grow and fade out .

5) Similarly, like the border spinners you can also customize the colors of growing spinners using the text color utility classes.

6) You can use the class .spinner-border-sm and .spinner-grow-sm to make a smaller 
spinner. Alternatively, you can use the custom CSS or inline styles to change the size as needed. 

7) You can easily align the spinners left, right or center using the flexbox, float, or text alignment utility classes.
